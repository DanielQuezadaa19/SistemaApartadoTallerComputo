Sistema web para la gestión y reserva de salas de cómputo.

UPG Apartados es una aplicación web desarrollada en PHP que permite a los docentes reservar salas de cómputo, visualizar disponibilidad en tiempo real y administrar sus reservas de manera sencilla.

El sistema calcula automáticamente:

Salas libres

Salas ocupadas

Reservas en proceso

Reservas finalizadas

También, desarrollaré un apartado de Administrador el cual podra tener acceso a todo el sistema, dar de alta-baja el estado de las computadoras. Podrá crear, eliminar editar y ver todos los usuarios.
